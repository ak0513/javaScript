요소(element) 사이즈 구하기

CSS

.box {width:200px;height:200px;padding:10px;border:3px solid #000;box-sizing: content-box;} .box-inner {margin:10px;padding:10px;border:3px solid #000;} .box-inner.box-scroll-x {width: 300px;} .box-inner.box-scroll-y {height: 300px;margin-top: 10px;}

element.offsetWidth
element.offsetHeight

  • padding, border 를 포함한 콘텐츠의 사이즈
  • margin 제외
margin - - - -
border 3 3 3 3
padding 10 10 10 10
200x200
var box = document.querySelector('.box'); box.offsetWidth // 226 = width(200) + padding(10+10) + border(3+3) box.offsetHeight // 226 = height(200) + padding(10+10) + border(3+3)

element.clientWidth
element.clientHeight

  • padding 을 포함한 콘텐츠의 사이즈
  • margin, border 제외
margin - - - -
border 3 3 3 3
padding 10 10 10 10
200x200
box.clientWidth // 220 = height(200) + padding(10+10) box.clientHeight // 220 = height(200) + padding(10+10)

element.scrollWidth
element.scrollHeight

  • padding 을 포함한 콘텐츠의 사이즈(자식요소 포함)
  • margin, border 제외
box-scroll-x w300
box-scroll-y h300
margin - - - -
border 3 3 3 3
padding 10 10 10 10
box-scroll
box.scrollWidth // 320 = padding(10+10) + box-scroll-x(300) box.scrollHeight // 385 = padding(10+10) + box-scroll-x(55) + margin(10) + box-scroll-y(300)

element.getBoundingClientRect().width
element.getBoundingClientRect().height

  • padding, border 를 포함한 콘텐츠의 렌더링 된 사이즈
  • margin 제외
  • 렌더링된 사이즈 반환
margin - - - -
border 3 3 3 3
padding 10 10 10 10
200x200
// offsetWidth, offsetHeight와 동일 box.getBoundingClientRect().width // 226 = width(200) + padding(10+10) + border(3+3) box.getBoundingClientRect().height // 226 = height(200) + padding(10+10) + border(3+3) // box에 transform:scale(50%); 적용 시 box.getBoundingClientRect().width // 113(transform:scale(50%)) box.getBoundingClientRect().height // 113(transform:scale(50%))
