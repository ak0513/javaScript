요소(element) 사이즈 구하기

.box {width:200px;height:200px;padding:10px;border:3px solid #000;box-sizing: content-box;} .box-inner {margin:10px;padding:10px;border:3px solid #000;} .box-inner.box-scroll-x {width: 300px;} .box-inner.box-scroll-y {height: 300px;margin-top: 10px;}

border-boxCSS

border-box 요소의 크기를 콘텐츠 영역과 패딩(padding), 그리고 테두리(border)를 모두 포함하여 계산합니다.
따라서 width와 height 속성이 콘텐츠 영역뿐만 아니라 패딩과 테두리까지의 크기를 의미하며, 콘텐츠 영역의 크기는 자동으로 조절됩니다.

offsetWidth / offsetHeight

offsetWidth는 DOM (문서 객체 모델)에서 사용되는 속성으로, 요소의 콘텐츠 영역을 나타냅니다.
이 너비에는 콘텐츠, 패딩(padding), 테두리(border) 영역을 포함하지만 여백(margin)과 세로 스크롤바는 제외됩니다.

margin - - - -
border 3 3 3 3
padding 10 10 10 10
200x200
border-box
content-box
box.offsetWidth // border-box인 경우 : 200 = width(200) // content-box인 경우 : 226 = width(200) + padding(10+10) + border(3+3) box.offsetHeight // border-box인 경우 : 200 = height(200) // content-box인 경우 : 226 = height(200) + padding(10+10) + border(3+3)

clientWidth / clientHeight

clientWidth는 DOM (문서 객체 모델)에서 사용되는 속성으로, 요소의 콘텐츠 영역을 나타냅니다.
이 너비에는 요소의 콘텐츠, 패딩(padding) 영역을 포함하지만, 테두리(border)와 세로 스크롤바는 제외됩니다.

margin - - - -
border 3 3 3 3
padding 10 10 10 10
200x200
border-box
content-box
box.clientWidth // border-box인 경우 : 194 = width(200) - border(3+3) // content-box인 경우 : 220 = height(200) + padding(10+10) box.clientHeight // 220 = height(200) + padding(10+10)

scrollWidth / scrollHeight

scrollWidth는 DOM (문서 객체 모델)에서 사용되는 속성으로, 요소의 콘텐츠 전체 영역을 나타냅니다.
이 너비는 요소의 콘텐츠, 패딩(padding) 영역을 포함하며, 만약 콘텐츠가 요소의 크기보다 크다면 스크롤링이 가능한 영역을 말합니다.

.box {width:200px;height:200px;margin-top: 20px;padding:10px;border:3px solid #000;box-sizing: content-box;} .box .box-inner {margin:10px;padding:10px;border:3px solid #000;background-color: #333;} .box-inner.box-scroll-x {width: 300px;height:50px;} .box-inner.box-scroll-y {height: 300px;}
box.scrollWidth // border-box인 경우 : 340 = padding(10+10) + box-scroll-x(300) + box-scroll-x margin(10+10) // content-box인 경우 : 366 = padding(10+10) + box-scroll-x(300) + box-scroll-x margin(10+10) + box-scroll-x padding(10+10) + box-scroll-x border(3+3) box.scrollHeight // border-box인 경우 : 400 = padding(10+10) + box-scroll-x(50) + box-scroll-x margin(10+10) + box-scroll-y(300) + box-scroll-y margin(10) // content-box인 경우 : 452 = // 20 = padding(10+10) + // 96 = box-scroll-x(50) + box-scroll-x margin(10+10) + box-scroll-x padding(10+10) + box-scroll-x border(3+3) + // 336 = box-scroll-y(300) box-scroll-y margin(10) + box-scroll-y padding(10+10) + box-scroll-y border(3+3)

getBoundingClientRect().width / getBoundingClientRect().height

getBoundingClientRect()은 DOM (문서 객체 모델)에서 사용되는 메서드로, 요소의 현재 뷰포트(viewport)에서의 위치와 크기 정보를 반환합니다.
이 메서드를 사용하면 요소가 화면 상에서 어떤 위치에 있고 어떤 크기를 가지는지를 알 수 있습니다.
