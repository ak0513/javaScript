요소 추가하기

prepend()

자바스크립트에서 prepend() 메서드는 DOM(Document Object Model) 요소의 내부에 새로운 자식 요소를 맨 앞에 추가하는 메서드입니다.
이를 통해 새로운 요소를 선택한 요소의 첫 번째 자식으로 삽입할 수 있습니다.

Syntax

prepend(param1) prepend(param1, param2) prepend(param1, param2, /* …, */ paramN)

예시

스크립트 적용 전
기존 자식요소
스크립트 적용 후
새로운 자식 요소
기존 자식요소
// 부모 요소를 선택합니다. const parentElement = document.getElementById('parent'); // 새로운 자식 요소를 생성합니다. const newChild = document.createElement('div'); newChild.textContent = '새로운 자식 요소'; // prepend() 함수를 사용하여 새로운 자식 요소를 부모 요소의 맨 앞에 추가합니다. parentElement.prepend(newChild);

append()

자바스크립트에서 append() 메서드는 DOM(Document Object Model) 요소의 내부에 새로운 자식 요소를 맨 뒤에 추가하는 메서드입니다.
이를 통해 새로운 요소를 선택한 요소의 마지막 자식으로 삽입할 수 있습니다.

Syntax

append(param1) append(param1, param2) append(param1, param2, /* …, */ paramN)

예시

스크립트 적용 전
기존 자식요소
스크립트 적용 후
기존 자식요소
새로운 자식 요소
// 부모 요소를 선택합니다. const parentElement = document.getElementById('parent'); // 새로운 자식 요소를 생성합니다. const newChild = document.createElement('div'); newChild.textContent = '새로운 자식 요소'; // append() 함수를 사용하여 새로운 자식 요소를 부모 요소의 마지막에 추가합니다. parentElement.append(newChild);

before()

자바스크립트에서 before() 메서드는 DOM(Document Object Model) 요소의 형제로 새로운 요소를 삽입하는 역할을 합니다.
이 메서드를 사용하면 선택한 요소의 이전 형제 요소로 새로운 요소를 추가할 수 있습니다.

Syntax

before(param1) before(param1, param2) before(param1, param2, /* …, */ paramN)

예시

스크립트 적용 전
기준 요소
스크립트 적용 후
새로운 형제 요소
기준 요소
// 기준이 될 요소를 선택합니다. const referenceElement = document.getElementById('reference'); // 새로운 형제 요소를 생성합니다. const newSibling = document.createElement('div'); newSibling.textContent = '새로운 형제 요소'; // before() 메서드를 사용하여 새로운 형제 요소를 기준 요소의 이전에 추가합니다. referenceElement.before(newSibling);

after()

자바스크립트에서 after() 메서드는 DOM(Document Object Model) 요소의 형제로 새로운 요소를 삽입하는 역할을 합니다.
이 메서드를 사용하면 선택한 요소의 이전 형제 요소로 새로운 요소를 추가할 수 있습니다.

Syntax

after(param1) after(param1, param2) after(param1, param2, /* …, */ paramN)
