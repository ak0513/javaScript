드래그 금지 테스트 영역입니다.

body { -webkit-user-select:none; -moz-user-select:none; -ms-user-select:none; user-select:none }

텍스트만 드래그를 허용하는 경우

.draggable { -webkit-user-select:text; -moz-user-select:text; -ms-user-select:text; user-select:text }
