문자열 찾기

indexOf

indexOf는 대부분의 프로그래밍 언어에서 사용되는 문자열 또는 배열에서 특정 요소나 문자열이 처음으로 나타나는 위치를 찾는 함수입니다.
주어진 요소나 문자열이 존재하지 않는 경우에는 보통 -1을 반환합니다.

Index 0 1 2 3 4 5
문자열 H e l l o w

문자열에서 특정 문자의 인덱스 찾기

const text = "Hello!"; const index = text.indexOf("e"); console.log(index); // 출력: 1

존재하지 않는 문자의 경우 -1 반환

const text = "Hello!"; const index = text.indexOf("x"); console.log(index); // 출력: -1
