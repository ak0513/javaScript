요소(element) 사이즈 구하기

offset

  • padding, border 를 포함한 컨텐츠의 사이즈
  • margin 제외

element.offsetWidth MDN

  • offsetWidth(226) = width(200) + padding(10+10) + border(3+3)

element.offsetHeight MDN

  • offsetHeight(226) = hegiht(200) + padding(10+10) + border(3+3)

client

  • padding 을 포함한 컨텐츠의 사이즈
  • margin, border 제외

element.clientWidth MDN

  • clientWidth(226) = width(200) + padding(10+10)

element.clientHeight MDN

  • clientHeight(226) = hegiht(200) + padding(10+10)

scroll

  • padding 을 포함한 컨텐츠의 사이즈
  • margin, border 제외

element.scrollWidth MDN

  • scrollWidth(320) = padding(10+10) + box-scroll-x(300)

element.scrollHeight MDN

  • scrollHeight(385) = padding(10+10) + box-scroll-x(55) + margin(10) + box-scroll-y(300)

getBoundingClientRect()

  • padding, border 를 포함한 컨텐츠의 사이즈
  • margin 제외

element.getBoundingClientRect().width MDN

  • scrollWidth(320) = padding(10+10) + box-scroll-x(300)

element.getBoundingClientRect().height MDN

  • scrollHeight(385) = padding(10+10) + box-scroll-x(55) + margin(10) + box-scroll-y(300)
